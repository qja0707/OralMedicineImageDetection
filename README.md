# OralMedicineImageDetection

## 프로젝트 개요
AI 모델을 활용하여 경구약제 객체를 탐지하는 프로젝트입니다. 이미지 데이터에서 최대 4개의 경구약제를 식별하고 위치를 검출하는 객체 탐지 모델을 구현과 성능 개선을 목표로 합니다.

## 프로젝트 목표
경구약제 이미지에서 객체를 정확하게 탐지할 수 있는 AI 모델을 구축하고, 이를 통해 약제 식별 자동화 가능성을 확인합니다.
